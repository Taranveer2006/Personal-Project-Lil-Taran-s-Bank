# Lil Taran's Bank

Lil Taran's Bank

Lil Taran's Bank is a browser-based banking simulator that I built using HTML, CSS, and JavaScript to practice front-end web development and DOM manipulation. The application allows users to view their account balance, deposit money, and withdraw money through an interactive interface.

Features
Display account balance.
Deposit money into the account.
Withdraw money from the account.
Real-time balance updates.
Toggle between Check Balance and Log Out states.
Deposit and Withdraw buttons are dynamically displayed after logging in.
Built using HTML, CSS, and JavaScript with event-driven programming.
Technologies Used
HTML5
CSS3
JavaScript (DOM Manipulation & Event Handling)
Purpose

This project was created to strengthen my understanding of JavaScript fundamentals, including event listeners, DOM manipulation, variables, functions, user interaction, and updating webpage content dynamically. It was my first banking simulator and served as the foundation for future improvements.

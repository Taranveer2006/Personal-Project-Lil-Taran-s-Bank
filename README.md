# Lil Taran's Bank

Lil Taran's Bank

Lil Taran's Bank is a browser-based simulator of the bank that I developed using HTML, CSS, and JavaScript to practice the front-end development and DOM manipulation. The simulation provides the functionality for viewing the balance of the account, depositing the money, and withdrawing the money.

Features View account balance. Deposit money in the account. Withdraw money from the account. Real-time update of the balance. Switch between the Check Balance and Log Out modes. Deposit and Withdraw buttons become visible after logging in. Developed with HTML, CSS, and JavaScript technologies using event-driven programming. Technologies Used HTML5 CSS3 JavaScript (DOM manipulation & event handling) Purpose

The purpose of the project was to enhance my skills in basic JavaScript operations such as creating events listeners, DOM manipulation, working with variables and functions, interacting with the user, and updating the webpage content.

Improvements that can be made to the project Multiple user authentication Accounts in different banks Check and Savings accounts Transfers of money between accounts Transaction history Storage of information using Local Storage and/or a database Use of forms rather than alerts AI bank assistant
